## Overview
The mini labs are a collection of exercises designed to teach and document core cybersecurity skills through hands-on experience.  
The focus is on understanding system behavior, Windows and Linux basics, logging, and early threat visibility. This section forms the foundation for more advanced detection, monitoring, and incident response exercises.

## Objectives
- Learn how to set up safe lab environments using virtual machines.
- Understand host vs guest systems and the importance of snapshots.
- Explore Windows system processes, user accounts, and services.
- Observe and analyze Windows logs, including successful and failed authentication events.
- Build early documentation skills for technical reporting.
